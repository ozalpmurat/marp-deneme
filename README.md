# marp-deneme

Bu proje, Marp kullanarak Markdown'dan sunumlar oluşturmak için bir örnek içerir.

## İçerik

- `deneme.md`: Örnek bir Marp sunumu dosyası.

## Nasıl Çalıştırılır

1. Marp CLI'yi yükleyin: `npm install -g @marp-team/marp-cli`
2. Sunumu HTML olarak dışa aktarın: `marp deneme.md --html`

## Lisans

Bu proje açık kaynaklıdır.
